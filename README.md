# 🛒 TechMart - E-Commerce Product Management System

TechMart is a Full Stack Java Spring Boot web application for managing products in an e-commerce store.

## Features

- Add Product
- View Products
- Edit Product
- Delete Product
- Search Products
- Sort Products
- Product Dashboard
- Product Details Page
- Responsive Bootstrap UI

## Technologies Used

- Java 21
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap 5
- Maven
- Git & GitHub

## Database

MySQL
